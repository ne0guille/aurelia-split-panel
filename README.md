**Installation**

`npm install aurelia-split-panel --save`

**Plugin Setup**
`aurelia.use.plugin(PLATFORM.moduleName('aurelia-split-panel'));`

**Usage**
``` <div split-panel>
      <div id="panel1">
        <p>ASDASd</p>
        <a>asdasd</a>
        <a>asdasd</a>
      </div>
      <div id="panel2">
        <p>ASDASd</p>
        <a>asdasd</a>
        <a>asdasd</a>
      </div>
    </div>
  ```
