**Installation**

`npm install aurelia-split-panel --save`

**Plugin Setup**
`aurelia.use.plugin(PLATFORM.moduleName('aurelia-split-panel'));`

**Usage**
``` 
<div split-panel>
    <div id="panel1">
      <p>test</p>
      <a>test</a>
      <a>test</a>
    </div>
    <div id="panel2">
      <p>test</p>
      <a>test</a>
      <a>test</a>
    </div>
  </div>
  ```
